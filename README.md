# 🧠 Deep Learning Lab Repository (`deep_learning_`)

Welcome to the **Deep Learning Lab Repository** 🚀
This repository contains all coursework, lab exercises, and projects completed as part of the Deep Learning curriculum.

---

## 📂 Repository Structure

### 🔹 1. CIA-1 (Continuous Internal Assessment - 1)

This section focuses on foundational neural network architectures:

* **Adaline Network**

  * Implementation of Adaptive Linear Neuron
  * Covers weight updates using gradient descent
  * Helps understand linear decision boundaries

* **Feed Forward Network**

  * Basic Artificial Neural Network (ANN)
  * Forward propagation and activation functions
  * Introduction to multi-layer networks

---

### 🔹 2. CIA-3 (Continuous Internal Assessment - 3)

This section explores advanced deep learning architectures:

* **CNN Architecture**

  * Implementation of Convolutional Neural Networks
  * Covers convolution, pooling, and fully connected layers
  * Used for image-based tasks

* **Next Word Predictor (LSTM)**

  * Sequence modeling using LSTM networks
  * Predicts the next word in a sentence
  * Demonstrates NLP concepts and text generation

---

### 🔹 3. End Trimester Examination Project

**📌 Disaster Prediction from Tweets**

* Uses deep learning to classify tweets as disaster-related or not
* Applies NLP techniques such as tokenization and embeddings
* Model built using LSTM for sequence understanding
* Real-world application of social media analysis

---

### 🔹 4. Final Consolidated Lab Manual

* A complete and well-structured document
* Covers all **7 lab exercises**
* Includes:

  * Theory explanations
  * Code snippets
  * Outputs and observations

---

### 🔹 5. Lab Exercises (1–7)

Step-by-step implementation of key deep learning concepts:

1. AdaLine and Feedforward Neural Network
2. Advanced Feedforward Neural Network
3. L1, L2 and Elastic Net Regulization Methods
4. Convolutional Neural Networks (CNN) and Real-time Object detection
5. Recurrent Neural Networks (RNN) and LSTM Networks
6. Sparse Autoencoders
7. Variable Autoencoders


   ```bash

   ```

3. Run any project:

   ```bash
   python main.py
   ```
